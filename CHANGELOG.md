# Changelog

All notable changes to the Cursor Monokai Theme will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.5.0] - 2024-11-20

### Added

- Initial release of Cursor Monokai Theme
- Comprehensive demo files showcasing syntax highlighting for multiple languages:
  - JavaScript, TypeScript, Python, Go, Rust, Ruby
  - React/JSX, HTML, CSS, Shell scripts, Markdown, JSON
- Full Cursor IDE optimization and branding

### Changed

- Renamed from "One Monokai" to "Cursor Monokai"
- Rebranded specifically for Cursor IDE
- Updated all documentation to focus on Cursor IDE

### Credits

This theme is based on the original [One Monokai Theme](https://github.com/azemoh/vscode-one-monokai) by [Joshua Azemoh](https://github.com/azemoh), which combines elements from Monokai and One Dark themes.
